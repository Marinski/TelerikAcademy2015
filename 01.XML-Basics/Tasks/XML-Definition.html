What does the XML language represent? What is it used for?

Extensible Markup Language (XML) is a markup language that defines a set of rules for encoding documents in a format which is both human-readable and machine-readable. XML is a software- and hardware-independent tool for storing and transporting data. It is defined by the W3C's XML 1.0 Specification and by several other related specifications, all of which are free open standards.
